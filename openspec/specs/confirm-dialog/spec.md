### Requirement: 확인 다이얼로그
시스템은 되돌릴 수 없는 작업 전에 사용자에게 확인을 요청하는 다이얼로그를 제공해야 한다.

#### Scenario: 확인 다이얼로그 표시
- **WHEN** 시스템이 확인 다이얼로그를 호출한다
- **THEN** 시스템은 오버레이와 함께 메시지, 확인 버튼, 취소 버튼을 표시해야 한다

#### Scenario: 확인 선택
- **WHEN** 사용자가 확인 버튼을 누른다
- **THEN** 시스템은 true를 반환하고 다이얼로그를 닫아야 한다

#### Scenario: 취소 선택
- **WHEN** 사용자가 취소 버튼을 누르거나 오버레이 배경을 클릭한다
- **THEN** 시스템은 false를 반환하고 다이얼로그를 닫아야 한다
