## MODIFIED Requirements

### Requirement: 관리자 로그인
시스템은 비밀번호 입력을 통한 관리자 로그인 기능을 제공해야 한다. 비밀번호는 서버 환경변수(ADMIN_PASSWORD)에 저장된 값과 비교하여 검증한다.

#### Scenario: 올바른 비밀번호 입력
- **WHEN** 사용자가 올바른 관리자 비밀번호를 입력하고 로그인 버튼을 누른다
- **THEN** 시스템은 관리자 모드를 활성화하고 관리자 전용 기능(멤버 삭제, MMR 수정)을 표시한다

#### Scenario: 잘못된 비밀번호 입력
- **WHEN** 사용자가 잘못된 비밀번호를 입력하고 로그인 버튼을 누른다
- **THEN** 시스템은 비밀번호가 올바르지 않다는 오류 메시지를 표시하고 관리자 모드를 활성화하지 않는다

#### Scenario: 비밀번호 미입력
- **WHEN** 사용자가 비밀번호를 입력하지 않고 로그인 버튼을 누른다
- **THEN** 시스템은 비밀번호 입력을 요구한다
