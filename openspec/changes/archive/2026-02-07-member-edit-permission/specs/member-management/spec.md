## MODIFIED Requirements

### Requirement: 멤버 정보 수정
시스템은 등록된 멤버의 이름, 주 포지션, 가능 포지션, 임시 여부를 누구나 수정할 수 있어야 한다. MMR 수정은 관리자로 로그인한 경우에만 가능하다.

#### Scenario: 비관리자 기본 정보 수정
- **WHEN** 비관리자가 멤버 카드를 클릭하고 이름, 포지션, 임시 여부를 변경한다
- **THEN** 시스템은 변경된 기본 정보를 저장한다

#### Scenario: 비관리자 MMR 수정 제한
- **WHEN** 비관리자가 멤버 수정 폼을 연다
- **THEN** MMR 필드는 비활성화되어 수정할 수 없다

#### Scenario: 관리자 MMR 수정
- **WHEN** 관리자가 로그인한 상태에서 멤버의 MMR을 직접 수정한다
- **THEN** 시스템은 입력된 MMR 값으로 업데이트한다

#### Scenario: 관리자 전체 필드 수정
- **WHEN** 관리자가 로그인한 상태에서 멤버 수정 폼을 연다
- **THEN** 모든 필드(이름, 포지션, MMR, 임시 여부)가 수정 가능하다
