## Why

사내/커뮤니티 LoL 내전을 진행할 때마다 수동으로 팀을 나누면 밸런스가 맞지 않고 시간이 오래 걸린다. 실력과 포지션을 고려해 자동으로 균형 잡힌 5:5 팀을 매칭하고, 내전 전적을 누적 관리하여 점점 더 정확한 밸런스 매칭이 가능한 웹 앱이 필요하다.

## What Changes

- 멤버 풀 관리 기능 (고정 멤버 등록/수정/삭제, 임시 참가자 추가)
- 멤버별 주 포지션 + 가능 포지션 목록 관리
- 내전 전적 기반 MMR 자동 산출 시스템
- 실력(MMR) + 포지션 밸런스를 고려한 5:5 자동 팀 매칭 알고리즘
- 경기 결과(승/패) 기록 및 전적 조회
- LoL 다크 테마 UI, 모바일 우선 반응형 디자인
- 탭 기반 네비게이션 (매칭 / 멤버관리 / 전적)
- JSON 파일 기반 로컬 데이터 저장

## Capabilities

### New Capabilities
- `member-management`: 멤버 풀 CRUD (이름, 주 포지션, 가능 포지션 목록, MMR). 고정 멤버 + 임시 참가자 지원
- `matchmaking`: 참가자 선택 → 실력(MMR) + 포지션 밸런스 기반 5:5 자동 팀 구성 알고리즘
- `match-history`: 경기 결과(승/패) 기록, 전적 조회, MMR 자동 업데이트
- `data-persistence`: JSON 파일 기반 로컬 데이터 저장/로드

### Modified Capabilities
(없음 - 신규 프로젝트)

## Impact

- **기술 스택**: TanStack Start (풀스택 프레임워크), Radix UI (컴포넌트), vanilla-extract (스타일링)
- **데이터**: JSON 파일 로컬 저장 (서버 불필요)
- **디자인**: LoL 다크 테마, 모바일 우선 (좁은 뷰), 탭 구분 (매칭/멤버/전적)
- **외부 의존성**: 없음 (독립 실행 웹 앱)
