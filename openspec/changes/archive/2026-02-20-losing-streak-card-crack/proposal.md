## Why

현재 연승 시에는 ShineBorder로 카드가 빛나는 효과가 단계별로 적용되지만, 연패 시에는 배지 텍스트("2연패", "3연패")만 표시될 뿐 시각적 피드백이 부족하다. 연패가 누적될수록 카드에 금이 가는(crack) 효과를 단계별로 적용하여 연승의 빛남 효과와 대칭적인 시각적 표현을 제공한다.

## What Changes

- 연패 횟수(2회 이상)에 따라 카드 360도 테두리에서 가운데를 향해 뻗어나가는 번개 모양 금 효과를 3단계로 적용
  - 2연패: 테두리 근처 짧은 금 5갈래 (가는 선, 낮은 불투명도)
  - 3연패: 중간까지 침투하는 금 8갈래, 굵어지고 가지 분기
  - 4연패 이상: 가운데까지 완전 관통하는 금 11갈래, 굵고 진한 표현
- 금 형태는 번개 모양(날카로운 지그재그), 시드 기반 절차적 생성으로 레벨별 일관된 패턴
- MemberCard (멤버 목록 페이지)와 SelectPhase (매치메이킹 선택 화면) 양쪽에 동일하게 적용
- 기존 연패 배지는 유지하면서 시각 효과를 추가
- 테두리 색상 변경 없음

## Capabilities

### New Capabilities
- `card-crack-effect`: 연패 횟수에 따른 3단계 카드 금 간 시각 효과. SVG 기반 번개 모양 오버레이를 시드 기반 절차적 생성으로 구현하며, streak count를 입력받아 적절한 crack 단계를 반환하는 유틸리티와 해당 스타일을 포함한다.

### Modified Capabilities
(없음 - 기존 streak 데이터 모델과 로직은 변경 없이 활용)

## Impact

- **UI 컴포넌트**: `MemberCard` (members-page.tsx), `SelectPhase` (select-phase.tsx)에 crack 오버레이 렌더링 추가
- **스타일**: `crack-overlay.css.ts` 파일에 오버레이 스타일 추가
- **의존성 변경 없음**: 순수 SVG + CSS 효과로 구현, 외부 라이브러리 불필요
- **데이터 모델 변경 없음**: 기존 `Streak` 타입의 `count`와 `type` 필드를 그대로 활용
