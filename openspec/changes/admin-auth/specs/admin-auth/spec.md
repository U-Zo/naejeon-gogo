## ADDED Requirements

### Requirement: 관리자 로그인
시스템은 비밀번호 입력을 통한 관리자 로그인 기능을 제공해야 한다. 비밀번호는 서버 환경변수(ADMIN_PASSWORD)에 저장된 값과 비교하여 검증한다.

#### Scenario: 올바른 비밀번호 입력
- **WHEN** 사용자가 올바른 관리자 비밀번호를 입력하고 로그인 버튼을 누른다
- **THEN** 시스템은 관리자 모드를 활성화하고 관리자 전용 기능(멤버 수정/삭제)을 표시한다

#### Scenario: 잘못된 비밀번호 입력
- **WHEN** 사용자가 잘못된 비밀번호를 입력하고 로그인 버튼을 누른다
- **THEN** 시스템은 비밀번호가 올바르지 않다는 오류 메시지를 표시하고 관리자 모드를 활성화하지 않는다

#### Scenario: 비밀번호 미입력
- **WHEN** 사용자가 비밀번호를 입력하지 않고 로그인 버튼을 누른다
- **THEN** 시스템은 비밀번호 입력을 요구한다

### Requirement: 관리자 로그아웃
시스템은 관리자가 로그아웃할 수 있어야 한다.

#### Scenario: 관리자 로그아웃
- **WHEN** 관리자가 로그아웃 버튼을 누른다
- **THEN** 시스템은 관리자 모드를 비활성화하고 관리자 전용 기능(멤버 수정/삭제 버튼)을 숨긴다

#### Scenario: 페이지 새로고침 시 로그아웃
- **WHEN** 관리자가 페이지를 새로고침한다
- **THEN** 시스템은 관리자 모드를 비활성화한다 (인증 상태 미보존)

### Requirement: 관리자 인증 상태 전역 관리
시스템은 React Context API를 통해 관리자 인증 상태를 앱 전역에서 접근 가능해야 한다.

#### Scenario: 인증 상태 전역 접근
- **WHEN** 관리자가 로그인한 상태에서 다른 탭(페이지)으로 이동한다
- **THEN** 관리자 인증 상태는 유지되어 모든 페이지에서 관리자 기능에 접근 가능하다

### Requirement: 관리자 로그인 UI
시스템은 관리자 로그인을 위한 UI를 제공해야 한다. 비밀번호 입력 필드와 로그인 버튼으로 구성된다.

#### Scenario: 비로그인 상태에서 로그인 버튼 표시
- **WHEN** 사용자가 관리자로 로그인하지 않은 상태이다
- **THEN** 레이아웃에 관리자 로그인을 위한 잠금 아이콘 버튼이 표시된다

#### Scenario: 로그인 버튼 클릭 시 비밀번호 입력 모달
- **WHEN** 사용자가 잠금 아이콘 버튼을 클릭한다
- **THEN** 비밀번호 입력 모달이 표시된다

#### Scenario: 로그인 상태에서 로그아웃 버튼 표시
- **WHEN** 관리자로 로그인한 상태이다
- **THEN** 잠금 아이콘 대신 잠금해제 아이콘 버튼이 표시되고, 클릭 시 로그아웃된다
